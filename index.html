exo使用总结

必备软件：

  • node – 这是一个 JavaScript 解释器

  • hexo (博客系统，是用 JavaScript 语言写的。npm install -g hexo)

  • git (这是一个版本控制系统)

  • hexo-deployer-git (hexo 的 git 部署插件。npm install –save hexo-deployer-git)

  • hexo deploy -g
